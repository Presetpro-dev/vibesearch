# Changelog

## 0.2.0 (unreleased)

Full SEO coverage. VibeSearch now spans the whole discipline; see docs/COVERAGE.md.

- New: `vibe-keywords` — intent-grouped keyword maps, hub-and-spoke clusters, cannibalization detection, striking-distance mining
- New: `vibe-links` — internal link map with paste-ready insertion list; honest backlink analysis from free exports
- New: `vibe-competitors` — verifiable side-by-side teardowns, gap lists, honest vs/alternatives page builder
- New: `vibe-data` — Search Console CSV/paste analysis with zero OAuth: striking distance, CTR gaps, decay, red flags
- New: `vibe-plan` — capacity-sized 90-day roadmap, every item mapped to its executing command, saved to .vibesearch/plan.md
- New: docs/COVERAGE.md — the discipline-by-discipline coverage and data-honesty map
- Router: ten commands, state-aware recommendation on bare /vibe
- Manifests bumped to 0.2.0
